# Sales Analyzer

Aplicativo em Python para consolidar múltiplas planilhas de vendas em Excel, limpar os dados automaticamente e gerar um relatório final com rankings e gráfico.

## Funcionalidades

- Leitura automática de múltiplos arquivos `.xlsx`
- Limpeza básica de dados inválidos
- Consolidação de vendas em uma única base
- Ranking por faturamento
- Ranking por quantidade
- Relatório final em Excel
- Gráfico automático no relatório
- Interface gráfica simples com Tkinter
- Botão para abrir o relatório gerado
